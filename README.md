# 🚀 Dockerized MERN Todo App

A full-stack MERN Todo Application built with React.js, Node.js, Express.js, MongoDB, Docker, and Docker Compose.

## 📌 Features

- Add new tasks
- Delete tasks
- MongoDB database integration
- REST API with Express.js
- Dockerized Frontend
- Dockerized Backend
- Docker Compose support

## 🛠️ Technologies Used

- React.js
- Node.js
- Express.js
- MongoDB
- Mongoose
- Docker
- Docker Compose

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/dockerized-mern-todo-app.git
```

Go to the project directory:

```bash
cd dockerized-mern-todo-app
```

Run the application:

```bash
docker compose up --build
```

## 🌐 Application URLs

Frontend:

```
http://localhost:3000
```

Backend:

```
http://localhost:5000
```

## 📡 API Endpoints

### Get all todos

```
GET /todos
```

### Add a todo

```
POST /todos
```

### Delete a todo

```
DELETE /todos/:id
```
